This is a static website built with only HTML and CSS, customized to resemble the look and feel of Harvard University’s website. I started from a basic template and modified it extensively to create a clean, responsive, and visually appealing design.

More About this Website:

Pure HTML & CSS used: No JavaScript or backend; fully static and lightweight

Custom Styling: Colors, typography, and layout adapted to emulate Harvard’s style while maintaining originality

Responsive Design: Layout adjusts gracefully to different screen sizes

Semantic HTML: Structured and organized for readability and maintainability

Portfolio-ready: A showcase of front-end design skills without relying on frameworks
